Лабораторная работа №2:

    Сортировка слиянием. Метод декомпозиции.

Цель лабораторной:

    Научиться писать эффективный алгоритм для сортировки - сортиртировку слиянием и поиска - бинарный поиск.

Задачи лабораторной:

    Научиться применять принцип "разделяй и властвуй",
    Написать, протестировать и понять принцип работы алгоритма сортировки слиянием,
    Написать, протестировать и понять принцип работы алгоритма бинарного поиска.

Структура проекта lab2:


    task1/ - первое задание.
        src/ — исходные коды.
        tests/ — тесты.

    task2/ - второе задание
        src/ — исходные коды.
        tests/ — тесты.

    task3/ - третье задание.
        src/ — исходные коды.
        tests/ — тесты.

    task4/ - четвертое задание.
        src/ — исходные коды.
        tests/ — тесты.

    task8/ - восьмое задание
        src/ — исходные коды.
        tests/ — тесты.
    
Тесты можно запустить из файлов в папках tests. Чтобы проверить работу программы в файле с самой программой, раскомментируйте вызов функции и запустите файл.
