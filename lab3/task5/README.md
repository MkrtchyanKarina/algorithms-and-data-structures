Функция `h_index(citations: list[int]) -> int` вычисляет индекс H (или H-index) для списка цитирований научных публикаций. H-index — это метрика, которая используется для оценки продуктивности и влияния ученого на основе количества его публикаций и количества цитирований этих публикаций.Запустить программу для текстового файла можно из точки входа, тесты можно запустить из соответствующей папки tests.
