Функция `worst_case(n: int) -> list[int]` создает список, который можно считать "худшим случаем" для алгоритма быстрой сортировки. То есть выбтрается pivot, который больше или меньше остальных элементов. Запустить программу для текстового файла можно из точки входа, тесты можно запустить из соответствующей папки tests.
