Задача №1
  Улучшение Quick sort

Реализация алгоритма быстрой сортировки (Quick Sort). Функция принимает список целых чисел `m` и сортирует его, разделяя на 3 списка: большие pivot, равные и меньшие. Запустить программу для текстового файла можно из точки входа, тесты можно запустить из соответствующей папки tests.
