Функция `scarecrow_sort` сортирует массив по "группам" с шагом `k` и проверяет, стал ли массив отсортированным после этого процесса. Возвращает "ДА" или "НЕТ" в зависимости от результата.Запустить программу для текстового файла можно из точки входа, тесты можно запустить из соответствующей папки tests.
