Задача №9

    Ближайшие точки

Программа реализует алгоритм для нахождения кратчайшего расстояния между парой точек в двумерном пространстве с использованием метода "Разделяй и властвуй".Запустить программу для текстового файла можно из точки входа, тесты можно запустить из соответствующей папки tests.
