Лабораторная 3 
        Быстрая сортировка, сортировки за линейное время

Цель лабораторной 
       Научиться применять алгоритм быстрой сортировки, а также алгоритмы линейной сортировки

Задачи лабораторной
        Научиться реализовывать и применять алгоритм быстрой сортировки
        Научиться применять линейные сортировки для специфических задач -например поразрядную сортировку
