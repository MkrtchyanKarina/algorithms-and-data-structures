Лабораторная работа №0. Введение.

Цели лабораторной:

    Научиться писать простые алгоритмы на Python. 
    Научиться оформлять проект и загружать его на github.com.

Задачи лабораторной:

    Вспомнить, как писать простые алгоритмы, как работать с файлами в Python. 
    Научться писать эффективные алгоритмы для последовательности Фибоначчи.
    Научиться оформлять проект, загружать его на github.com.
    Научиться писать отчёт по лабораторной работе.

Структура проекта lab0:
    
    task1/ - первое задание.
        src/ — исходные коды.
        tests/ — тесты.

    task2/ - второе задание
        src/ — исходные коды.
        tests/ — тесты.

    task3/ - третье задание.
        src/ — исходные коды.
        tests/ — тесты.
       
Тесты можно запустить из файлов в папках tests. Чтобы проверить работу программы в файле с самой программой, раскомментируйте вызов функции.
